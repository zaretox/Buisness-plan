# Mon Système — app

App à 4 onglets : Système · Setup · Contenu · Offres.

## Mettre en ligne sur GitHub (depuis le navigateur)
1. Crée un dépôt sur github.com (bouton + → New repository), nom au choix, en **Public**.
2. Add file → **Upload files** → glisse le contenu de ce dossier (au moins `index.html`) → Commit.
3. Settings → **Pages** → Branch : `main` → Save. Au bout d'1 min tu as une adresse
   du type `https://tonpseudo.github.io/nom-du-depot/`.

## Modifier le code
Ouvre `index.html` sur GitHub → icône crayon (✏️) → modifie → Commit.
L'app se met à jour toute seule au prochain chargement.

## L'installer comme RECOMP sur iPhone
Ouvre l'adresse **dans Safari** → bouton Partager → **Sur l'écran d'accueil** → Ajouter.
Elle s'ouvre en plein écran comme une app. Les cases cochées sont gardées sur le téléphone.

Tout le contenu est dans `index.html` — c'est le seul fichier à éditer.
